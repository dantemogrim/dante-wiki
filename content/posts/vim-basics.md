### Copy

- Current line: `yy`

### Delete

- Current line: `dd`
- Two words: `d2w`
- Unwanted character: `x`

### Exit

- `:q`
- All instances: `:qa`
- Without saving: `:q!`

### Open

File: `$ vim target.txt`
Vimtutor: `$ vimtutor`

### Modes

- Insert: `I`
- Normal: `<Esc>`

### Navigation

- Bottom: `L`
- Down: `j`
- Left: `h`
- Right: `l`
- Top: `H`
- Up: `k`

### Paste

- `p`

### Replace

- Current character: `r` + `<desired character>`

### Save

- And exit: `:wq`
- Current file: `:w`
- Current file as: `:sav filename.ext`

### Select

- All: `ggVG`

### Undo

- `u`
